TradingAgents — Arquitectura del Sistema

Vista técnica del flujo de ejecución: desde la entrada del usuario hasta la decisión final, incluyendo capas de datos, orquestación LangGraph, agentes LLM y gestión de estado.

Framework: LangGraph
LLM: OpenAI / Anthropic / Google
Data: yfinance / Alpha Vantage
Indicadores: stockstats
Validación: Pydantic
Cache: CSV local
Versión: 0.2.2

REFERENCIA DE COMPONENTES

Orquestación — LangGraph

  • graph/setup.py TradingAgentsGraph
  • graph/conditional_logic.py routing condicional
  • graph/propagation.py extracción de resultados
  • graph/reflection.py aprendizaje iterativo
  • graph/signal_processing.py extrae rating final
  • Estado compartido via AgentState TypedDict

Estado del Agente

  • agents/utils/agent_states.py
  • company_of_interest ticker analizado
  • trade_date fecha de análisis
  • market_report output analista mercado
  • fundamentals_report output analista fundamentales
  • news_report output analista noticias
  • sentiment_report output analista sentimiento
  • investment_debate_state rondas de debate
  • risk_debate_state debate de riesgo
  • final_trade_decision decisión final

Capa de Datos — Routing

  • dataflows/interface.py router central
  • Prioridad 1: tool_vendors[method]
  • Prioridad 2: data_vendors[category]
  • Fallback solo en AlphaVantageRateLimitError
  • Categorías: core_stock_apis, technical_indicators, fundamental_data, news_data
  • Default: todo a yfinance

Cálculo de Indicadores

  • dataflows/stockstats_utils.py
  • dataflows/y_finance.py
  • Librería: stockstats.wrap(df)
  • Trigger lazy: acceso a df[indicator]
  • Pre-procesamiento: ffill().bfill(), NaN drop, timezone strip
  • Cache: CSV local 15 años de historia
  • Retry: backoff exponencial 2s→4s→8s
  • 13 indicadores: SMA50/200, EMA10, MACD, RSI, Bollinger, ATR, VWMA

Nodos Analistas

  • agents/analysts/market_analyst.py
  • agents/analysts/fundamentals_analyst.py
  • agents/analysts/news_analyst.py
  • agents/analysts/social_media_analyst.py
  • LLM: quick_thinking_llm (gpt-5-mini)
  • Herramientas via llm.bind_tools()
  • Tool schemas de Annotated + @tool
  • El LLM decide hasta 8 indicadores relevantes

Debate y Decisión

  • agents/researchers/bull_researcher.py
  • agents/researchers/bear_researcher.py
  • agents/managers/research_manager.py
  • agents/trader/trader.py
  • agents/risk_mgmt/ (3 perfiles)
  • agents/managers/portfolio_manager.py
  • LLM: deep_thinking_llm (gpt-5.2)
  • Config: max_debate_rounds=1
  • Config: max_risk_discuss_rounds=1
  • Output: BUY / OVERWEIGHT / HOLD / UNDERWEIGHT / SELL

Configuración

  • tradingagents/default_config.py
  • llm_provider: openai / anthropic / google
  • deep_think_llm: gpt-5.2
  • quick_think_llm: gpt-5-mini
  • data_cache_dir: ./dataflows/data_cache
  • results_dir: ./results
  • max_recur_limit: 100
